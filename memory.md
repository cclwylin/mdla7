# MDLA7 Memory: High-Level Module Spec

This file is the short memory version. For detailed ports/descriptors, see
`spec.md`.

## Global Rules

- Final target is FPGA.
- Active RTL tree is `rtl/verilog`; do not recreate `verilog_ctrl` /
  `verilog_final`.
- Final Verilog dataflow must stay:
  `DRAM -> UDMA -> L1 -> Engine/Postprocess -> L1 -> UDMA -> DRAM -> check`.
  For CONV, postprocess means `CONV psum/acc -> Requant -> L1`.
- Engines must not secretly read files or bypass Host / Command / UDMA /
  L1Manager / L1Mesh control.
- `--option dpi` may accelerate arithmetic only; it must not bypass the real
  control/data path.
- Verilog mode should be more RTL than CX mode. CX may model abstract route/resource
  timing; Verilog targets should implement explicit hardware structure such as
  arbiters, FIFOs, routers, links, and handshakes.
- Current engine coverage is mostly sample/tilelet sized. Do not tune
  synth/verilog cycle ratio until compact tile/full traversal exists.

## Module Bus

| Block / Link                  | Drawio bus groups                                         |
| ----------------------------- | --------------------------------------------------------- |
| Host -> Command Engine        | descriptor `1T`                                         |
| Command Engine -> each engine | config `W (1T)`                                         |
| CONV ACT read                 | `32 x 16B R`, direct L1Mesh -> CONV                     |
| CONV WGT read                 | `32 x 16B R`, direct L1Mesh -> CONV                     |
| CONV writeback                | no target Payload W to L1; psum/acc must chain to Requant |
| CONV -> Requant chain         | dedicated `4096 bit/cycle` INT32/acc chain              |
| Requant                       | `8 x 16B R` + `8 x 16B W`                             |
| EWE                           | `16 x 16B R` + `8 x 16B W`                            |
| POOL                          | `16 x 16B R` + `8 x 16B W`                            |
| TNPS                          | `8 x 16B R` + `8 x 16B W`                             |
| L1Manager <-> L1Mesh          | `16 x 16B R` + `16 x 16B W`                           |
| UDMA <-> L1Manager            | `16 x 16B R` + `16 x 16B W`                           |
| UDMA <-> DRAM                 | 4 x 16B AXI R/W                                           |
